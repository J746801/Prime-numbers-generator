# Prime-numbers-generator
A super fast prime numbers generator that uses the Sieve of Eratosthenes method. I tested and it can generate prime numbers up to 100 million in just over 11 seconds!
Use prime_generator_sieve_of_Eratosthenes.py to generate prime numbers. Simply enter the number you want it to generate to on line 40 and run the file, then use getPrimes.py (just an example) to use the prime numbers.
